- (String) attrs_to_string(ops, keys = nil)

Builds a string of attributes suitable for HTML

There’s likely a better more-standard way to do this. (Method is only used by the #table method.)

Examples:

With all keys:
  hash={:foo'=>'bar',:goo'=>'car',;hoo=>'dar'}
  attrs(hash) => ' foo="bar" goo="car" hoo="dar"'

With selected keys:
  hash={:foo'=>'bar',:goo'=>'car',;hoo=>'dar'}
  attrs(hash,[:foo,:hoo]) => ' foo="bar" hoo="dar"'

Options Hash (keys):

  • (Array) select N/A — only certain attributes to use in generated string

Returns:

  • (String) — attributes suitable for HTML

def attrs_to_string(ops,keys=nil)
 return '' if !ops or ops=={}
 keys||=ops.keys
 return keys.inject(''){|s,k| s = (ops[k] ? s+=" #{k}=\"#{ops[k]}\"" : s)}
end

- (String) table(ops = {})

Processes parameters into a complete HTML table

Examples:

Build a table with three data rows, each with three columns, plus a header and footer row,
giving the table both an ID and class:

  headers = ['a','b','c']
  footers = ['x','y','z']
  rows=[['d','e,'f'],['g','h,'i'],['j','k','l']]
  table(:id=>'foo', :class=>'bar', :headers=>headers, :footers=>footers, :rows=>rows)

Gets this HTML
  <table id="foo" class="bar">
  <thead>
  <tr>
  <th>a</th>
  <th>b</th>
  <th>c</th>
  </tr>
  </thead>
  <tbody>
  <tr>
  <td>d</td>
  <td>e</td>
  <td>f</td>
  </tr>
  <tr>
  <td>g</td>
  <td>h</td>
  <td>i</td>
  </tr>
  <tr>
  <td>j</td>
  <td>k</td>
  <td>l</td>
  </tr>
  </tbody>
  <tfoot>
  <tr>
  <th>x</th>
  <th>y</th>
  <th>z</th>
  </tr>
  </tfoot>
  </table>

Special Cases

If headers, row or footers are nil, then they're skipped.

If footers==[] then this method will use a footers array
with blanks that is the same length as the headers array.

Parameters:

  • (String) text — the complete text of the table, e.g. text
  • (Array, String) headers — builds the header row from this array
  • (Array, String) footers — builds the footer row from this array
  • (Array, Array) rows — builds the data rows from this array of arrays
  • (String) class — is the table’s css class, e.g. :class=>’sortable’
  • (String) id — is the table’s css id, e.g. :id=>’mytable’

Returns:

  • (String) — complete HTML table

def table(ops={})
  text=ops[:text]
  if !text
   headers=ops[:headers]
   footers=(defined?(ops[:footers]) ? ((ops[:footers]==true) ? Array.new(headers.size,'') : ops[:footers]) : false)
   rows=ops[:rows]
   text=((headers ? thead(headers) : '') + (rows ? tbody(rows) : '') + (footers ? tfoot(footers) : ''))
  end
  return "<table#{attrs_to_string(ops,[:id,:class])}>\n" + text + "</table>\n"
end

def wrap(text,tag)
  t=tag
  t.sub!(/^</,'')
  t.sub!(/>$/,'')
  open=t
  shut=t.split.first
  "<#{open}>#{text}</#{shut}>"
end
